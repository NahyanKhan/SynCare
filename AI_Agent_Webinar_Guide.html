Section 05

Step-by-Step Build Guide

Follow these steps during the live webinar. Each step is written so clearly that a complete beginner can follow along without getting lost.

Create a Free n8n Cloud Account

Go to n8n.io and click "Get Started Free." Sign up with your email. You'll land on the n8n dashboard — this is where all your automations live. Think of it as your personal automation studio. No downloads, no installations — it runs entirely in your browser.

Get Your Free OpenRouter API Key

Go to openrouter.ai, click "Sign In" and use Google login. Once inside, go to API Keys → click Create Key. Name it "Study Assistant". Copy the key — it looks like: sk-or-xxxxxxxxxxxx. This key is what allows n8n to talk to the AI. Keep it safe, don't share it.

Create Your Telegram Bot

Open Telegram on your phone. Search for @BotFather (the official bot creator). Send the message /newbot. Give your bot a name like "MyStudyBot" and a username like "mystudyhelper_bot". BotFather will reply with a token that looks like: 7123456789:AAHxxxxxxxxxxxxxxxx. Copy this token — it's your bot's password.

Create a New Workflow in n8n

In n8n, click the big "+ New Workflow" button. You'll see a blank canvas — like a blank whiteboard. Give your workflow a name: "AI Study Assistant". This is where you'll drag and connect all the building blocks of your agent.

Add the Telegram Trigger Node

Click the "+" button on the canvas. Search for "Telegram Trigger". Select it and drag it onto the canvas. Click on it to open its settings. Under Credentials, click "Create New" and paste your Telegram Bot Token. Set the trigger to fire on "Message". This node will "wake up" your workflow every time someone sends your bot a message.

Add an HTTP Request Node (AI Connector)

Click "+" to add another node. Search for "HTTP Request". This node can talk to any API on the internet — including OpenRouter's AI. Set the method to POST and paste this URL:

https://openrouter.ai/api/v1/chat/completions

Under Headers, add: Authorization: Bearer YOUR_API_KEY_HERE and Content-Type: application/json.

Configure the AI Request Body

In the HTTP Request node, go to Body → choose JSON. Paste this template — it tells the AI how to behave and what to answer:

{ "model": "mistralai/mistral-7b-instruct:free", "messages": [ { "role": "system", "content": "You are a friendly AI study assistant for college students. Explain concepts clearly and simply. Keep answers under 200 words. Use examples." }, { "role": "user", "content": "{{ $json.message.text }}" // ← This pulls the student's message } ] }

The magic part is {{ $json.message.text }} — this automatically picks up whatever the student typed in Telegram.

Add a "Set" Node to Extract the AI Reply

Add a "Set" node after the HTTP Request. This node plucks out just the AI's reply from the full response. Create a new field called aiReply with this expression:

{{ $json.choices[0].message.content }}

This navigates into the AI response to grab the actual text of the answer. Without this step you'd be sending the student a giant wall of code instead of a clean answer.

Add the Telegram "Send Message" Node

Add a "Telegram" node (not the trigger, the action one). Connect it to your bot credentials (same token as before). Set the Chat ID to: {{ $('Telegram Trigger').item.json.message.chat.id }}. Set the message text to: {{ $json.aiReply }}. This sends the AI's answer back to whoever asked the question.

Activate & Test Your AI Agent

Connect all nodes in order (Telegram Trigger → HTTP Request → Set → Telegram). Click the big blue "Activate" toggle in the top right. Now open Telegram, find your bot, and send it: "What is Newton's second law?". Watch it reply with a real AI-generated answer in seconds! 🎉

📄 AI PDF Summarizer

Upload any PDF (lecture notes, research paper, textbook chapter) to a Google Drive folder. n8n watches the folder, extracts the text, sends it to the AI, and messages you a clean bullet-point summary on Telegram.

🧪 AI Exam Prep Quiz Bot

Tell the bot your exam topic and how many questions you want. It generates a multiple-choice quiz, waits for your answers one by one, scores you at the end, and explains each wrong answer. Like a personal tutor.

Connect your Gmail. When you receive an email from a professor, forward it to your Telegram bot with a quick instruction like "Write a polite reply asking for an extension." The AI drafts the reply — you just review and send.

New skills learned: Gmail node, email parsing, professional tone prompting.
